## 🔥 Features

| Category       | Commands / Capabilities                                                                 |
|----------------|------------------------------------------------------------------------------------------|
| **🎨 Media**   | Sticker maker (image/video), video to MP3, to GIF, rotate, flip, slowmo, circle crop, trim, black video |
| **📥 Downloads** | Instagram, Facebook, TikTok, Pinterest, YouTube (audio & video), Spotify search          |
| **🤖 AI**      | Gemini AI (chat & question answering), song recognition (Shazam), image recognition       |
| **🎮 Fun**     | Death Note quotes (KIRA), anime search, GIF search, math game, word chain, tic‑tac‑toe   |
| **🛠️ Utility** | Weather (4‑day forecast), movie info, lyrics, translation, Wikipedia, variables, AFK     |
| **👑 Admin**   | Group management: kick, add, promote, demote, mute, warn, antilink, welcome, goodbye     |

- [Node.js](https://nodejs.org/) (v20+)
- [FFmpeg](https://ffmpeg.org/download.html) – add to PATH or place `ffmpeg.exe` in bot folder
- [Git](https://git-scm.com/)

### Steps

git clone https://github.com/Madhavgkmd/kira-md-bot.git
